Database Configuration

Connect Firebase Firestore (free tier) to sync menu data across all devices.

Firebase Setup Guide

  1. Go to Firebase Console and create a free project
  2. Click Build > Firestore Database and create a database (start in test mode)
  3. Go to Project Settings > General > Your apps and add a Web app
  4. Copy the Firebase config values and paste them above
  5. Click Save & Connect
